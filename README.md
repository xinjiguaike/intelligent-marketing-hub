# AI 智能体营销系统

## 项目概述

这是一个基于 AI 智能体技术的智能营销系统，将传统的"工具型平台"转变为"智能组织型平台"。通过多个具身智能体扮演品牌营销团队中的各类角色，实现从策划、创作、运营到洞察的全流程智能化营销工作。

## 核心特性

### 1. 多角色智能体系统

系统包含 5 个核心智能体角色：

- **🎯 营销总监 (Chief Marketing Agent)** - 全局协调、任务分配、审批决策
- **📝 策划专家 (Planning Agent)** - 策略制定、方案规划、市场分析
- **✍️ 创意团队 (Creative Agents)** - 文案创作、视觉设计、素材制作
- **📢 运营专家 (Operation Agent)** - 投放执行、账号管理、KOL 对接
- **📊 数据分析师 (Analyst Agent)** - 数据监测、效果分析、优化建议

### 2. 人机协同模式

系统支持多种协同模式：

- **AI 独立模式**：AI 智能体自主完成任务
- **人工模式**：真人独立执行任务
- **协同模式**：AI 生成初稿，真人审核修改
- **辅助模式**：真人主导，AI 提供建议和参考

### 3. 任务流转机制

```
创建任务 → 总监分配 → 角色执行 → 提交审核 → 通过/退回 → 流转至下游
           ↓                                          ↓
      可指定 AI/真人                              可退回修改
```

### 4. 导航与空间概览

| 导航空间 | 关键能力 | 说明 |
| --- | --- | --- |
| **个人工作台** | 多角色视角切换 | 快速查看当前任务、通知与个人日程 |
| **智能任务编排** | 任务脉搏、AI/人工协同看板 | 统一管理营销任务的全流程状态 |
| **营销策划空间** | 策略模板、AI 共创流程 | AI 与策划组合的多阶段策划管线 |
| **创意工作室** | 图文/海报/视频 AI 创作 | 支持素材同步、版本比对与 Workflow 联动 |
| **数字员工管理** | 运行指标、团队管理、训练建议 | 管理数字员工的能力、任务和协作团队 |
| **运营中心** | 跨渠道营销日历、排期甘特、KOL/KOC 看板 | 聚合营销行为与投放节奏的运营驾驶舱 |
| **洞察中心** | 实时监控、AI 思维链、复盘报告 | 数据洞察与智能分析入口 |
| **品牌资源中心** | 账号、达人、商品、内容资产管理 | 品牌资源的一体化管理与引用 |
| **AI 智能体中控** | 数字员工运行态、指令队列、协同日志 | 观察与调度 AI 智能体的中枢 |
| **系统偏好设置** | 智能协同、通知、安全策略 | 管理 AI 行为边界与通知机制 |
| **知识库** | 策略模板、执行手册、草稿审阅、标签管理 | 统一沉淀与运营营销知识资产 |

## 快速开始

### 前端 Demo

前端 Demo 已完成，使用 Mock 数据模拟所有交互。

```bash
cd ai-intelligent-marketing-hub
cd frontend
npm install
npm run dev
```

访问 `http://localhost:5173` 查看效果。

### 功能演示

1. **切换角色**：点击右上角用户头像，切换不同的营销角色视图
2. **创建任务**：在营销总监视图中创建营销任务，分配给不同角色
3. **AI 协同**：在各角色工作台中体验 AI 生成、人工优化的协同流程
4. **任务流转**：观察任务如何在不同角色之间流转和处理

### 演示场景

系统内置"双十一家电促销活动"完整营销场景：

1. 总监创建活动策划任务 → 分配给策划专家
2. 策划专家使用 AI 生成策略方案 → 人工优化后提交审批
3. 总监审批通过 → 任务流转至文案师
4. 文案师 AI 生成多组文案 → 选择优化后传递给设计师
5. 运营专家（AI）自动制定投放计划 → 关键节点人工确认
6. 数据分析师（AI）实时监测 → 自动生成分析报告

更多体验建议：

- **数字员工管理**：在“数字员工管理”中查看在线状态、任务排队，并尝试添加自定义数字员工或人机协作团队。
- **知识库**：在“知识库”中添加新的策略条目、草稿或标签，体验知识资产的沉淀和审阅流程。
- **AI 智能体中控**：查看指令队列与协同日志，理解 AI 的调度链路。

## 技术架构

### 前端技术栈

- **框架**：React 18 + TypeScript + Vite
- **UI 组件库**：Arco Design
- **状态管理**：Zustand
- **路由**：React Router v6
- **流程可视化**：React Flow (规划中)
- **动画**：Framer Motion (规划中)

### 后端技术栈 (规划中)

- **框架**：FastAPI (Python 3.11+)
- **智能体编排**：LangGraph
- **LLM 集成**：LangChain
- **AI 服务**：OpenAI GPT-4 / Claude 3.5
- **数据库**：PostgreSQL + Redis
